# Sistema de Gestión Hospitalaria V4.0

Proyecto final de programación en lenguaje C. Sistema integral para la administración de citas médicas, control de personal y gestión de pacientes.

## Integrantes
- Isaac Danilo Valencia Pillalaza
- Ronald Sebastian Sisa Anacleto

## Características
- **Asignación Automática:** Algoritmo inteligente que busca médicos disponibles por especialidad.
- **Roles de Usuario:** Módulos diferenciados para Administradores, Médicos y Pacientes.
- **Persistencia de Datos:** Uso de archivos de texto (.txt) organizados en carpetas.
- **Seguridad:** Login sin claves maestras y validación de usuarios.

## Instrucciones para Ejecutar
1. Descargar el código o clonar el repositorio.
2. Compilar el archivo `main.c` (incluyendo `citas.c`, `medicos.c`, `pacientes.c`, `admin.c`).
3. **IMPORTANTE:** Antes de ejecutar, asegurarse de que existen las siguientes carpetas en el directorio:
   - `db/pacientes`
   - `db/medicos`
   - `db/admins`
   - `db/registros`
   - `reportes`
4. Ejecutar el programa `.exe` generado.
